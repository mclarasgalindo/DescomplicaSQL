O que são?

As funções agregadas são usadas para realizar cálculos sobre um conjunto de valores e retornar apenas um resultado. Normalmente, elas são aplicadas junto com a cláusula GROUP BY, que permite organizar os dados em grupos com base em uma ou mais colunas.

Mostra tudo junto na mesma linha para cada tipo de embalagem. No SQL, existem cinco funções de agregação principais, amplamente utilizadas em todos os bancos de dados. Cada uma delas executa um tipo específico de operação sobre os conjuntos de dados e são fundamentais para análises e resumos de informações.

COUNT

Esta função é usada para contar o número de linhas em uma coluna específica ou no conjunto de resultados. É útil para saber quantos registros atendem a um certo critério.

    SELECT embalagem escolhe a coluna embalagem

    COUNT(*) as contagem →conta quantos registros existem para cada tipo de embalagem e dá o nome de contagem a essa coluna.

    FROM tabela_de_produtos → diz de qual tabela os dados estão sendo retirados.

    GROUP BY embalagem → agrupa os resultados de acordo com cada tipo de embalagem.

SUM

Como o nome sugere, a função SUM é usada para somar valores de uma coluna. É extremamente útil em situações financeiras, como calcular o total de vendas ou despesas.

    O comando SUM retorna a soma dos valores do campo preço por embalagem.

AVG

AVG (Média): Esta função calcula a média dos valores em uma coluna. É amplamente utilizada em análises estatísticas, como encontrar a média de preços, idades ou qualquer outro valor numérico.

    SELECT embalagem escolhe a coluna embalagem

    AVG(preço) as média →calcula a média aritmética dos preços dos produtos daquele tipo de embalagem e dá o nome de média à coluna.

    FROM tabela_de_produtos → indica de onde virão os dados.

    GROUP BY embalagem → agrupa os registros por embalagem, para que a média seja calculada para cada grupo.

MAX e MIN:
