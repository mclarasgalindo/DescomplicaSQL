Comandos DDL

O que é um DDL?

Em bancos de dados, DDL significa Linguagem de Definição de Dados (Data Definition Language). É um conjunto de comandos SQL usados para definir e gerenciar a estrutura de um banco de dados, incluindo a criação, alteração e remoção de objetos como tabelas, índices, visualizações, etc.

CREATE

Usado para criar objetos no banco (tabelas, bancos de dados, visões, índices, etc.).

ALTER

Serve para modificar a estrutura de um objeto existente, como adicionar ou remover colunas.

DROP

Remove completamente um objeto do banco. Cuidado: apaga a estrutura e todos os dados dentro dela.
